# AI PC Manager

一个基于AI的PC管理员智能体，可以通过自然语言执行系统管理任务。

## 功能特点

- 自然语言输入
- 智能命令转换
- 多步骤任务支持
- 上下文管理
- 错误处理

## 安装

```bash
pip install -e .
```

## 使用方法

直接运行命令：

```bash
ai-pc-manager
```

## 开发

1. 创建虚拟环境：

```bash
# 使用 miniconda 创建虚拟环境
conda create -n apm python=3.11
conda activate apm
```

2. 安装开发依赖：

```bash
pip install -r requirements-dev.txt
```

3. 安装项目依赖：

```bash
pip install -e .
```
